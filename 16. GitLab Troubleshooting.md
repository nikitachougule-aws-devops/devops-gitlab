## 1. Pipeline is Stuck in Pending

**Problem:**
The pipeline starts but remains in the **Pending** state.

**Reason:**

- No GitLab Runner is available.
- Runner is offline.
- Runner is not assigned to the project.

**Solution:**

- Check Runner status.
- Register or start a Runner.
- Ensure the Runner is assigned to the project.

---

## 2. Pipeline Failed

**Problem:**
The pipeline shows **Failed**.

**Solution:**

- Open **Build → Pipelines**.
- Select the failed job.
- Check the job logs.
- Fix the error and push the changes again.

---

## 3. `.gitlab-ci.yml` Syntax Error

**Problem:**
Pipeline is not created because of an invalid YAML file.

**Solution:**

- Check indentation.
- Verify keywords.
- Validate the YAML file using GitLab's CI Lint tool.

---

## 4. Job Failed

**Problem:**
A specific job fails.

**Solution:**

- Read the job logs.
- Check script commands.
- Verify required tools and dependencies are installed.

---

## 5. Merge Request Pipeline Not Running

**Problem:**
Creating a Merge Request does not start a pipeline.

**Solution:**

- Check the pipeline rules in `.gitlab-ci.yml`.
- Ensure Merge Request pipelines are enabled.

---

## 6. Permission Denied

**Problem:**
Pipeline cannot access a file or execute a command.

**Solution:**

Give execute permission if needed.

```bash
chmod +x script.sh
```

Verify project and runner permissions.

---

## 7. Docker Image Not Found

**Problem:**

```text
Image not found
```

**Solution:**

- Verify the image name.
- Verify the image tag.
- Ensure the image exists in the registry.

---

## 8. Authentication Failed

**Problem:**

```text
Authentication failed
```

**Solution:**

- Check username and password.
- Use a Personal Access Token instead of a password.
- Verify registry credentials.

---

## 9. Variables Not Available

**Problem:**
A CI/CD variable is empty.

**Solution:**

- Verify the variable name.
- Check project or group variables.
- Ensure Protected variables are used only on protected branches.

---

## 10. Artifact Not Found

**Problem:**
A job cannot find the artifact from a previous job.

**Solution:**

- Verify the artifact path.
- Ensure the artifact was created.
- Check that it has not expired.

---

## 11. Cache Not Working

**Problem:**
Dependencies are downloaded every pipeline.

**Solution:**

- Verify the cache path.
- Ensure the cache configuration is correct.
- Check if the cache key has changed.

---

## 12. Push Rejected

**Problem:**

```text
failed to push some refs
```

**Solution:**

Pull the latest changes before pushing.

```bash
git pull origin main
git push
```

---

## 13. Merge Conflict

**Problem:**
GitLab cannot merge the branch automatically.

**Solution:**

- Pull the latest changes.
- Resolve the conflicts locally.
- Commit and push the resolved code.

---

## 14. Runner Offline

**Problem:**
The Runner is not available.

**Solution:**

Check the Runner status.

```bash
sudo gitlab-runner status
```

Restart if necessary.

```bash
sudo gitlab-runner restart
```

---

## 15. Deployment Failed

**Problem:**
The deploy stage fails.

**Solution:**

- Check deployment logs.
- Verify server connectivity.
- Verify SSH keys and deployment credentials.
- Ensure deployment scripts are correct.

---

| Problem | Solution |

| Pipeline Pending | Start or register a GitLab Runner |
| Pipeline Failed | Check pipeline logs and fix errors |
| YAML Error | Fix `.gitlab-ci.yml` syntax |
| Job Failed | Check job logs and scripts |
| MR Pipeline Not Running | Verify pipeline rules |
| Permission Denied | Check file permissions (`chmod +x`) |
| Docker Image Not Found | Verify image name and tag |
| Authentication Failed | Use valid credentials or access token |
| Variables Missing | Check CI/CD Variables configuration |
| Artifact Missing | Verify artifact path and expiration |
| Cache Not Working | Check cache configuration |
| Push Rejected | `git pull` then `git push` |
| Merge Conflict | Resolve conflicts and push again |
| Runner Offline | Restart or register the Runner |
| Deployment Failed | Check deployment logs and credentials |
