## Goal

By the end of these notes, you will be able to:

- Work with GitLab repositories
- Create and manage Merge Requests
- Build CI/CD pipelines
- Configure GitLab Runners
- Use variables, artifacts, cache, and registry
- Troubleshoot pipeline issues
- Build and deploy applications using GitLab CI/CD
- Use AI to improve development and DevOps workflows
